pyLoudness
===============================

version number: 0.0.1
author: Jason Rigden

Overview
--------

Get the loudness of a audio file.

Installation / Usage
--------------------

To install use pip:

    $ pip install pyloudness


Or clone the repo:

    $ git clone https://github.com/jrigden/pyloudness.git
    $ python setup.py install
